# Ratehub Affiliate Widgets Directory

**Last Updated**: Q4 2025

This document provides a complete inventory of all Ratehub widgets available for affiliate partners to embed on their websites. Each widget can be self-hosted or embedded via a Ratehub-hosted subdomain.

---

## Quick Reference

### Widget Hosting Options

Widgets can be hosted in two ways:

1. **Ratehub-Hosted Subdomain** (Recommended)
   ```
   https://<partner-domain>.ratehub.ca/<BU>/<product-name>
   ```
   Example: `https://redflagdeals.ratehub.ca/mortgages/affordability-calculator`

2. **Partner-Hosted** (Direct Embedding)
   - Widget code provided for direct embedding
   - Design and copy customization available upon request

## Integration Guide

### For Widget Embedding

All available widgets can be embedded using one of these methods:

#### Method 1: Ratehub Subdomain (Recommended)
```
https://<partner-domain>.ratehub.ca/<bu>/<product-name>

Examples:
- https://redflagdeals.ratehub.ca/mortgages/affordability-calculator
- https://redflagdeals.ratehub.ca/banking/tfsa-calculator
- https://redflagdeals.ratehub.ca/loans/debt-consolidation-calculator
```

**Advantages:**
- No technical integration required
- Automatic updates and maintenance
- Mobile-responsive by default
- Affiliate tracking built-in
- Design consistency with Ratehub

#### Method 2: Direct Embed Code
- Available for selected widgets
- Custom styling options available
- Contact Ratehub for specific embed code

### Customization Options

For most widgets, we can adjust:
- ✅ Design (colors, layout)
- ✅ Copy (text content, headings)
- ✅ Branding (logos, colors)
- ✅ Affiliate tracking parameters

**Request customizations** by contacting partnerships@ratehub.ca

### Common Integration Questions

**Q: How do I embed a widget on my site?**  
A: Use the Ratehub subdomain method (`https://<partner>.ratehub.ca/...`). No code needed—it's a simple iframe that we handle.

**Q: Can I customize the widget design?**  
A: Yes, for most widgets. Contact partnerships@ratehub.ca with your requirements.

**Q: How do I track affiliate clicks?**  
A: Affiliate tracking is automatic when using the Ratehub subdomain method. You can also provide a partner parameter.

**Q: What if a widget I want isn't available?**  
A: Some widgets (CardFinder, LoanFinder) are restricted for compliance. We recommend using the subdomain redirect alternative or choosing a similar available widget.
